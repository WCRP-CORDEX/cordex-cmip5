Public CORDEX-CMIP5 documents and lists for referencing in web sites

This website has been updated to the new CORDEX nomenclature after CORDEX-CMIP6. The contents have not been updated and are kept as originally maintained at https://github.com/IS-ENES-Data/IS-ENES-Data.github.io. As such, this documentation refers to the CORDEX experiment to downscale CMIP5 global models (CORDEX-CMIP5) simply as CORDEX.
